# 🚀 COMMENT DÉPLOYER SUR VERCEL (Gratuit)

## Étape 1: Créer un compte Vercel
1. Ouvre **Chrome** sur ton téléphone
2. Va sur: **https://vercel.com**
3. Clique **"Sign Up"**
4. Choisis **"Continue with GitHub"** (le plus simple)
5. Si tu n'as pas GitHub → utilise ton email

## Étape 2: Mettre ton code sur GitHub

### Option A: Via GitHub Mobile (Recommandé pour toi)
1. Télécharge l'app **GitHub** sur ton téléphone
2. Connecte-toi avec ton compte
3. Créé un nouveau repo: **AvatarForge**
4. Clique **"Add file"** → **"Create new file"**
5. Copie-colle le contenu de chaque fichier:
   - `src/App.tsx` → crée le fichier `src/App.tsx`
   - `index.html` → crée le fichier `index.html`
   - `vite.config.ts` → crée le fichier `vite.config.ts`
6. Commit (en bas de l'écran)

### Option B: Via GitHub Website
1. Va sur **https://github.com**
2. Créé un repo `AvatarForge`
3. Clique **"Add file"** → **"Create new file"**
4. Nomme le fichier `src/App.tsx`
5. Copie tout le code de `src/App.tsx`
6. Répète pour chaque fichier

## Étape 3: Déployer sur Vercel
1. Sur **Vercel**, clique **"Add New..."** → **"Project"**
2. Sélectionne **"Import Git Repository"**
3. Trouve ton repo `AvatarForge`
4. Clique **"Deploy"**
5. Attends 1-2 minutes
6. **C'est FAIT !** Tu as ton lien 🔗

# 📱 COMMENT POSTER SUR TIKTOK

## 1. Prépare ta Vidéo
1. Génère la vidéo avec D-ID ou SadTalker
2. Télécharge sur ton téléphone
3. Ouvre **CapCut** (gratuit sur Android)
4. Ajoute la vidéo
5. Ajoute de la musique (sons trending)
6. Ajoute des effets si tu veux
7. Exporte en haute qualité

## 2. Poste sur TikTok
1. Ouvre TikTok
2. Clique **"+"** pour créer
3. Upload ta vidéo
4. Écris une description accrocheuse
5. Ajoute les hashtags:
   ```
   #fyp #pourtoi #viral #astuce #motivation
   ```
6. Ajoute cover (image de preview)
7. Poste !
